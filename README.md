# Run scripts 🚀
- Step 1: Open command prompt terminal in the project folder
- Step 2: Create virtual enviroment
  ```python
  py -3.12 -m venv venv 
  ```
- Step 3: Open virtual enviroment
  ```python
  venv\Scripts\activate
  ```
- Step 4: Install required dependencies
  ```python
  pip install -r requirements.txt
  ```
- Step 5: Install torch
  ```python
  pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
  ```
- Step 6: Run the test_py.py for testing
  ```python
  python ./tests/test_gpu.py
  ```
  Make sure that the CUDA is supported and the GPU name is displayed
